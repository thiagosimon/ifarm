# LGPD Data Incident

## Detection

- Any unauthorized access to personal data
- Data leak in logs, public endpoints, or third-party systems

## Immediate (0-24h)

1. Contain the breach: revoke access, patch vulnerability
2. Identify affected data and users
3. Notify DPO (dpo@ifarm.com.br)
4. Preserve evidence (logs, access records)

## Notification (24-72h)

1. DPO assesses severity
2. If ANPD notification required (within 72h):
   - Prepare notification template
   - Include: nature of incident, affected data categories, number of affected users, measures taken
3. Notify affected users if high risk

## Post-Incident

- Root cause analysis
- Implement corrective measures
- Update security controls
- Document in incident register
